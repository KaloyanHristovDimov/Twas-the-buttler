# 'Twas the Butler

## Overview

**'Twas the Butler** is a mobile murder mystery deduction game where the player takes the role of a student at a detective academy. Each level represents one of the student's practical exams, where they must investigate a staged crime scene and solve the case before time runs out.

The game focuses on observation, clue collection, logical deduction, and timed decision-making. Every test takes place in the same room, which has been designed to resemble a murder scene. While the setting remains consistent, each level changes the victim, murderer, clues, weapon, and motive, creating a new case for the player to solve.

The project is currently planned to include **three levels**, with each case becoming progressively more difficult and complex.

---

## Game Concept

The player is a detective academy student undergoing a series of crime scene investigation tests. In each level, they are placed inside a staged crime scene and must identify the correct solution within a time limit.

The player must explore the room, inspect clues, connect evidence, and determine key details of the case, such as:

- Who was murdered
- Who the murderer is
- What weapon was used
- What motive led to the crime

The challenge increases with every level by introducing more clues, misleading evidence, and more complicated relationships between characters.

## Platform

**Target platform:** Mobile

The game is designed specifically for mobile devices, with interactions and UI planned around touch controls.

---

## Gameplay

The core gameplay loop is based around investigation and deduction:

1. The player enters the crime scene.
2. They inspect the room and interact with clues.
3. Important evidence is collected or recorded.
4. The player uses the gathered information to solve the case.
5. The player submits their final answer before the timer runs out.

Each level is structured as a different detective academy test, allowing the game to reuse the same room while changing the case details and investigation logic.

## Visual Style

The intended visual direction combines a cozy but suspicious detective-room atmosphere with stylized mystery game elements.

The moodboards focus on:

- Warm wooden interiors
- Detective boards and investigation notes
- Classic mystery-room furniture
- Clue-focused objects and props
- A slightly dramatic but playful detective tone
- 2D character and UI inspiration from mystery/adventure games
- A compact room-based investigation setup suitable for mobile

The game aims for a readable, atmospheric style that supports investigation gameplay without overwhelming the player visually.

## Current Scope

The current goal is to create a playable mobile prototype with three complete levels.

Each level will use the same main crime scene room but will include different:

- Characters
- Victim and murderer
- Murder weapon
- Motive
- Clues
- Difficulty level

The first level will introduce the player to the basic investigation mechanics, while later levels will require more careful observation and deduction.
